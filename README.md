# fisiotheapp challenge
Challenge from DevChallenge. The objective was creating an app with two screens using the files obtained on the following link:
[fisiotheapp - desing folder](https://github.com/Lorenalgm/fisiotheapp/tree/master/design)

## About the repository
The app was made with kotlin. The app is **not** complete. My goal with this project was to train handling fragments and navigation.  


# Explaining the screens 

### First Screen 

First screen has a start button that sends you to the second screen so the user can start the treatment,
the bottom navigation view is functional, so it just needs to be directed to other layouts.

# desafio fisiotheapp
Desafio da DevChallenge. O objetivo era criar um app com duas telas usando os arquivos contidos no seguinte link:

[fisiotheapp - pasta desing](https://github.com/Lorenalgm/fisiotheapp/tree/master/design)

## Sobre o repositório
O app foi feito com kotlin. O aplicativo **não** está completo. Minha meta com este projeto era treinar a manutenção de fragments e navigation.


# Explicando as telas

### Primeira tela

Primeira tela tem um botão para começar o tratamento, que te envia para a segunda tela, a bottom navigation view está funcional, então só é preciso adicionar os layouts ao nav_graph.
